## How to Run

1.	Clone this repo:

```
git clone https://github.com/your-username/random-password-generator.git
cd random-password-generator
```

2.	Install dependencies:

```
pip install streamlit
```

3.	Launch the app:

```
streamlit run app.py
```

4.	Open the app in your browser:

```
http://localhost:8501
```
